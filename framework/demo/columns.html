Responsive Flex Columns

This grid system uses Flexbox wrapping with gap. Column widths are calculated dynamically to account for the gap using:
calc((100% - (cols - 1) * gap) / cols).
Resize the browser to see breakpoints (1100px, 768px, 479px).
